Open Source

I love Open Source. Like most devs, I cut my teeth pulling apart OS codebases. Here's some of the code I've released to the world.

kMailQueue

kMailQueue is a simple but powerful mail queue system for Kohana 3.x.

Configurable batch sizes, security features and a dead simple API.

MAuth

MAuth is a flexible authentication library, again for Kohana 3.x.

Per-user permissons, multiple instances and ORM agnostic.

My first open-source jQuery plugin

The plugin mimics the HTML5 placeholder attribute across any browser consistently.
